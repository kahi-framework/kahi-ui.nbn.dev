Card

Card is a multi-part surface pattern for displaying a section of content in a long-form vertical format.

Imports

<script>
    import {Card} from "@kahi-ui/framework";

    const {
        Container,
        Figure,
        Footer,
        Header,
        Section,
    } = Card;
</script>

Palette

You can change the color palette of the Card via the palette property.

Elevation

You can set how "high" your Card will appear to be over top the page via the elevation property.

Flush

You can change the appearance of the Card to be flush with the rest of the Application content.

Properties

Card.Container

elevation

Description

Alters how "high" the Card appears to be off the page.
Types
lowest (DEFAULT) low medium high highest

palette

Description

Alters the displayed color scheme.
Types
auto inverse inherit accent dark light alert affirmative negative

variation

Description

Alters the appearance of the Card.
Types
flush
