Official

Below you can find resources made maintained by Kahi UI contributors.

Discussion / Support

Official support provided by contributors and community members.

  • Issue Tracker — Official GitHub issue tracker for filing bugs and requesting support.

Showcase

Projects that show off the Kahi UI framework.

  • kahi-framework/kahi-ui.nbn.dev — Official documentation site.

Templates

Projects that can be used as a basis for your own project, showing how to integrate with various software.

  • kahi-framework/kahi-ui-template-svelte — Official template for Svelte based on npx degit sveltejs/template my-svelte-project starter.
  • kahi-framework/kahi-ui-template-sveltekit — Official template for SvelteKit based on the npm init svelte@next starter.
