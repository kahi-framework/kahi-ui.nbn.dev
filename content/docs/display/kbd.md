# Kbd

> **NOTE**: New since `v0.6.0`.

`Kbd` is typically used to communicate a keybinding to the user.

```svelte {title="Kbd Preview" mode="repl"}
<script>
    import {Kbd} from "@kahi-ui/framework";
</script>

<Kbd>CTRL+C</Kbd>
```
